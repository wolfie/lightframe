LightFrame Project Page

LightFrame currently lacks a proper webpage, and probably will continue to be without one for a while (due to development time constraints). This is why I set up this little page to give an overview of the different resources being used during the LightFrame development.

The sites are:

http://lightframe.org/

These pages. This will be changed once a proper 'community website' has been constructed. This includes a manual, installation files, and probably some kind of discussion forum.

http://blog.lightframe.org/

Random thoughts. A place where I (and, for the moment, only I – Hopefully that'll change...) jot down my thoughts that might relate to LightFrame, the development thereof, or maybe of above.

The best source for status reports on LightFrame, definitely.

http://repo.lightframe.org/

Hot cakes. This is how LightFrame's source looks like at any given moment. This is the official LightFrame source repository, and it's a free-for-all.

http://tickets.lightframe.org/

Todos. A mix of bug reports, upcoming features, "how about this?":es and the occasional unintelligible mumbling that, at the time, seemed very important to write down for the future me. Without this, LightFrame would've probably be ditched ages ago.

Documentation

The user manual in its present state. This is what I'm working on currently, and one of the most important components for the community site. The manual is written in slightly modified Markdown, so don't be surprised if it doesn't render properly with a normal parser.
